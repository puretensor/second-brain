# Briefing Note Template

## Format

```markdown
# Daily Briefing -- YYYY-MM-DD

## Priority Items
- [item requiring immediate attention]
- [item requiring immediate attention]

## Calendar
- [today's events from ops@puretensor.ai calendar]

## Email Summary
- [unread count per account]
- [flagged/urgent items]

## Project Updates
- [active project status changes]
- [pending items approaching deadlines]

## Infrastructure
- [cluster health summary]
- [any alerts or anomalies]

## Action Items
- [ ] [specific actionable item]
- [ ] [specific actionable item]
```

## Guidelines
- Lead with what needs attention, not what's fine
- Numbers over narratives
- Flag deadlines within 48 hours
- Cross-reference pending items from memory.md
